# Dyn

Dynamic array, iterable, list, and map utilities with good support for Typescript.

## Usage

```typescript
import * as D from "@asmartbear/dyn"

...
```


## Development

Build:

```bash
npm run build
```

Unit tests:

```bash
npm run test
```

Unit tests, refreshed live:

```bash
npm run watch
```

Prepare for release (e.g. run tests and bump version number), and then release:

```bash
npm run release && git push --follow-tags origin main && npm publish
```
